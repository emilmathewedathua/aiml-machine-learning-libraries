This repository contains my solutions for Machine learning-libraries assignment from the AI/ML course.
The assignment focuses on practicing commonly used Python libraries for machine learning and data analysis.

The work was done using Google Colab and Python.

Libraries Used

NumPy – for array operations

Pandas – for data handling and analysis

Matplotlib – for basic data visualization

Seaborn – for advanced visualization

Tasks Performed
NumPy:

Created a 5x5 array with random numbers between 1 and 50

Found the row with the maximum sum

Replaced elements greater than the mean with 0

Created an array from 1 to 20 and selected every 3rd element starting from the 2nd element

Pandas:

Loaded the Iris dataset

Displayed the first 10 rows of the dataset

Generated basic statistical information

Filtered rows based on conditions

Grouped the data by species and calculated mean petal length

Data Processing

Added a new column called petal_ratio (petal_length / petal_width)

Identified the top 5 rows with the highest petal ratio

Data Visualization

Using Matplotlib:

Histogram of sepal length

Scatter plot of sepal length vs petal length

Using Seaborn:

Pairplot of the entire dataset

Boxplot showing petal width distribution for each species

Dataset
-------

The Iris dataset is used in this assignment. It is a well-known dataset used for practicing machine learning and data analysis techniques.
